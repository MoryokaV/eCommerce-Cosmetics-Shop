# eCommerce Cosmetics Shop

A simple cosmetics shop app for placing orders online.

# Technologies

Dart + Flutter Framework => UI  
Sqflite (Dart package) => Sqlite backend  
Python => Testing / Learning Sqlite databases + REST API (Flask)
